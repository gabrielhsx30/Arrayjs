Exercícios de Java Script

Exercício 1

Criar um algoritmo que receba 10 números e exiba todos eles na tela. Use um array de números para auxiliar a executar o algoritmo.

Exercício 2

Criar um algoritmo que receba 10 números e exiba todos eles na tela, mostrando quantos são pares e quantos são ímpares.Use um array de números para auxiliar a executar o algoritmo.
